This index ensures standardization across all Agents, Artifacts, and Documentation.

## Key Terms

- **CTZ/CTZD**: Compact Time Zulu formats for timestamps and dates. See [[timestamp-formats]].
- **FRAG**: A specific task unit. See [[frag-pkg-system]].
- **PKG**: A collection of related FRAGs. See [[frag-pkg-system]].
- **Lynchpin**: A mandatory completion condition. See [[frag-pkg-system]].
- **Vulnerability**: A failure condition to be avoided. See [[frag-pkg-system]].
- **LTM/STM**: Long-Term and Short-Term Memory layers. See [[memory-architecture]].
- **Firmware**: Operational logic directives. See [[memory-architecture]].
- **Manual**: The LLM's mirror of code logic. See [[memory-architecture]].
