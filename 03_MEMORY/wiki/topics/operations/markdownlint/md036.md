# MD036 - Emphasis used instead of a heading

**Summary**: This rule identifies instances where emphasized (bold or italic) text is used to separate sections, recommending proper Markdown headings instead.

**Sources**: `tmp/MD036.md`

**Last updated**: 2026-05-26

---

This check looks for instances where emphasized (i.e. bold or italic) text is used to separate sections, where a heading should be used instead:

```markdown
**My document**

Lorem ipsum dolor sit amet...
```

To fix this, use Markdown headings (`#`, `##`, etc.) instead of emphasized text to denote sections.

Rationale: Using emphasis instead of a heading prevents tools from inferring the structure of a document, affecting accessibility and semantic parsing.
