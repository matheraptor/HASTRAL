# Operational Conventions

**Summary**: The standardized operational framework for HASTRAL agents and documentation.
**Sources**: (source: CONVENTIONS.md)
**Last updated**: 2026-04-16

---

HASTRAL utilizes a set of strict conventions to ensure consistency across all agents, artifacts, and documentation within the MAGPIE ecosystem.

## Core Operational Pillars

- **Operational Doctrine**: The core hierarchy of **DOCTRINE => OPERATION => MISSION => TASK => STEP**. Defined in [[/CONVENTIONS.md]].
- **Memory Architecture**: Structured layers of persistence. See [[]].
- **Task Management**: Missions as standalone packages. See [[]].
- **Timekeeping**: Standardized timestamps and dates. See [[../../operations/temporal/timestamp_formats]].
- **Validation**: The 5-STEPS priority hierarchy. See [[]].
