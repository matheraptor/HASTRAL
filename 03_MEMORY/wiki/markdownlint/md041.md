# md041

**Summary**: First line in a file should be a top-level heading.

**Sources**: (source: doc/Rules.md)

**Last updated**: 2026-05-15

---

This rule is triggered when the first line in a document is not a top-level (h1) heading.

> [!WARNING]
> **Frontmatter Exception**: If the document contains YAML frontmatter, this rule is deferred. The top-level heading must immediately follow the closing frontmatter delimiter. See [[markdownlint/md061]] for the specific structural requirement.
