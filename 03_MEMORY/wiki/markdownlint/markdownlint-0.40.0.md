# markdownlint-0.40.0

**Summary**: A Node.js style checker and lint tool for Markdown/CommonMark files.

**Sources**: (source: README.md)

**Last updated**: 2026-05-15

---

Markdown is easy to read, write, and understand, but its flexibility can lead to inconsistent formatting. `markdownlint` is a static analysis tool for Node.js that uses a library of rules to enforce standards and consistency. It uses the `micromark` parser and honors the CommonMark specification, while supporting GitHub Flavored Markdown (GFM) features.

It was inspired by Mark Harrison's `markdownlint` for Ruby.

## Key Features

- Enforces formatting standards for Markdown files.
- Uses `micromark` for parsing and honors CommonMark specification.
- Supports GFM (autolinks, tables, directives, footnotes, math).
