# md001

**Summary**: Heading levels should only increment by one level at a time.

**Sources**: (source: doc/Rules.md)

**Last updated**: 2026-05-15

---

This rule is triggered when you skip heading levels in a Markdown document. Nested headings should increase by only one level at a time.

## Examples

### Bad
```markdown
# Heading 1

### Heading 3
```

### Good
```markdown
# Heading 1

## Heading 2

### Heading 3
```
