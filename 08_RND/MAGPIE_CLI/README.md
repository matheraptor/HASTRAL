# MAGPIE CLI + Agent

MAGPIE CLI is a command-line interface + agent framework for [MAGPIE](https://github.com/matheraptor/MAGPIE.git), a tool for analyzing and visualizing protein-protein interaction networks.

## Installation

```bash
pip install magpie-cli
```

## Usage

```bash
magpie --help
```
